Installation
============

Die Installation von Yii beinhaltet hauptsächlich diese beiden Schritte:

   1. Herunterladen des Yii-Frameworks von [yiiframework.com](http://www.yiiframework.com/).
   2. Entpacken der Yii-Version in ein Verzeichnis mit Webzugriff.

> Tip|Tipp: Yii muss nicht zwingend in ein Verzeichnis mit Webzugriff installiert
werden. Eine Yii-Anwendung verwendet ein einzelnes Eingangsscript. In der
Regel ist dies die einzige Datei, die für Web-Benutzer zugänglich sein muss.
Andere PHP-Scripts, inklusive denen von Yii, sollten vor Zugriff über das Web
geschützt werden, da sie für Hack-Versuche missbraucht werden könnten.

Anforderungen
-------------

Nachdem Sie Yii installiert haben, möchten Sie vielleicht überprüfen,
ob Ihr Webserver alle Anforderungen von Yii erfüllt. Dazu können Sie
eine Prüfung über diese URL in einem Webbrowser ausführen:

~~~
http://hostname/pfad/zu/yii/requirements/index.php
~~~

Die Minimalanforderungen von Yii ist, dass Ihr Webserver PHP 5.1.0 oder höher
unterstützt. Yii wurde mit dem [Apache HTTP server](http://httpd.apache.org/) 
unter Windows- und Linux-Betriebssystemen getestet. Es läuft u.U. auch auf
anderen Webservern und Plattformen, vorausgesetzt PHP 5 wird dort unterstützt.
