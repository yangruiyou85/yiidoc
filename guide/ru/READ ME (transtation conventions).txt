ПЕРЕВОД ТЕРМИНОВ

action	действие
Active Record	без перевода
attach handler	"назначить обработчик" или "связать обработчик с событием"
attribute of the model	атрибут модели
controller	контроллер
extension	дополнение
field (of the table)	поле (или атрибут) таблицы
getter	метод чтения
handler (event)	обработчик (события)
method	метод (объекта). Внимание! У объета/класса нет функций, есть только методы
model	модель
model form	модель формы
~note	без перевода	//Обсуждается
parameter	параметр (у метода или функции, никак не у класса)
property	свойство (объекта)
resolve request	предварительная обработка запроса
row (of the table)	строка (таблицы)
setter	метод записи
~tip	без перевода	//Обсуждается
view	представление	

ГРАММАТИКА:
- слова "ихние" нет, есть слово "их"
- в слове "атрибут" после "а" только одна буква "т"
- вводные слова (например, пожалуйста, кстати и т.п.) выделяются запятыми
- "тся" и "ться" проверяется так. Допустим, слово "появится". Вопрос к слову: "Что сделает?" После "т" нет мягкого знака - значит, без него. А вот если правильный по контексту вопрос: "Что сделать?", мягкий знак есть, и будет писаться "появиться". Но смысл меняется. Будьте внимательны
- (Добавить правила оформления списков).

ПРОЧЕЕ:
- повествование в соответствии с оригиналом ведется от первого лица мноножественного числа (мы);
- комментарии в коде тоже переводятся;
- если есть что добавить по сути к документации - добавляем, но пишем "(прим. перев.)" (примечание переводчика)
